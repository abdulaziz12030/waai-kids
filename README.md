# واعي كيدز | WAAI KIDS

منصة تربوية ذكية تجمع الأسرة والطفل والمعلم في رحلة واحدة لبناء العادات الإيجابية، وتحقيق الأهداف، ومتابعة حفظ القرآن، وتحفيز الإنجاز بالمكافآت الهادفة.

## الهوية المعتمدة

- الاسم العربي: واعي كيدز
- الاسم الإنجليزي: WAAI KIDS
- النطاق الأساسي: `waai-kids.com`
- اسم حزمة التطبيق: `waai-kids`

## التقنيات

- Next.js
- React
- TypeScript
- Supabase
- Vercel

## التشغيل المحلي

```bash
npm install
npm run dev
```

ثم افتح `http://localhost:3000`.

## متغيرات البيئة

يحتاج المشروع إلى متغيرات اتصال Supabase المعتمدة في بيئة التطوير والنشر. لا تُحفظ المفاتيح السرية داخل المستودع.

## النشر

الفرع الإنتاجي هو `main`، ويُنشر المشروع تلقائياً عبر Vercel. النطاق النهائي المعتمد للمنصة هو `waai-kids.com`.
